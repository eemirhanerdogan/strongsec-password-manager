# STRONGSEC - Chrome Password Manager

STRONGSEC is a privacy-focused Chrome extension designed to securely manage passwords directly in the browser.

## Features

- Strong password generator
- Smart autofill for login forms
- Secure local vault
- Save credentials after signup
- Multi-account support
- Breach detection
- Master password protection
- Auto-lock secure sessions

## Security

- AES-GCM local encryption
- PBKDF2 key derivation
- No plaintext password storage
- Local-first security architecture
- Privacy-preserving breach detection

## Project Structure

- `password-extension/` → Chrome extension source code
- `strongsec-website/` → Landing page source code
